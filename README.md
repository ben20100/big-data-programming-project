# ⚽ 유럽 축구 경기 데이터 기반 승패 요인 분석

## 1. 문제 정의

현대 축구에서는 단순한 승패 결과뿐만 아니라 점유율, 슈팅 수, 패스 성공률, 선수 기여도 등 다양한 데이터가 경기 결과에 영향을 미친다. 하지만 이러한 데이터는 규모가 크고 다양한 형태로 존재하기 때문에 효율적인 저장 및 분석 시스템이 필요하다.

본 프로젝트에서는 유럽 축구 경기 데이터를 활용하여 경기 승패에 영향을 미치는 요인을 분석하고자 한다. 특히 홈 경기 여부, 점유율, 슈팅 수 등의 요소가 실제 경기 결과와 어떤 관계를 가지는지 분석한다.

이를 위해 공개 축구 데이터를 수집하고 Hadoop 기반 빅데이터 처리 환경(HDFS, Hive, Spark)을 활용하여 데이터 저장, 전처리, 분석 및 시각화를 수행한다.

---

## 2. 주요 분석 질문

### 1) 홈/원정에 따라 승률 차이가 있는가?

축구 경기에서는 일반적으로 홈 경기에서 유리하다는 “홈 어드밴티지(Home Advantage)”가 존재한다고 알려져 있다.

본 프로젝트에서는 팀별 홈 승률과 원정 승률을 비교하여 실제 데이터에서도 이러한 경향이 나타나는지 분석한다.

#### 분석 요소
- 홈 경기 승률
- 원정 경기 승률
- 팀별 홈/원정 성적 비교
- 리그 전체 평균 비교

---

### 2) 점유율이 높으면 실제 승률도 높은가?

점유율이 높은 팀이 경기를 지배한다고 평가되지만, 실제 승리와 반드시 연결되는지는 명확하지 않다.

본 프로젝트에서는 경기 점유율 데이터와 실제 경기 결과를 비교하여 점유율과 승률 간의 상관관계를 분석한다.

#### 분석 요소
- 평균 점유율과 승률 비교
- 점유율 대비 득점 효율
- 점유율이 높지만 패배한 경기 사례 분석
- 상관계수(Correlation) 분석

---

### 3) 어떤 스탯이 승리에 가장 큰 영향을 미치는가?

축구 경기에는 슈팅 수, 유효 슈팅, 패스 성공률, 파울 수 등 다양한 경기 스탯이 존재한다.

본 프로젝트에서는 여러 경기 데이터를 기반으로 승패와 가장 관련성이 높은 스탯을 분석한다.

#### 분석 요소
- 슈팅 수와 승률 관계
- 유효 슈팅 대비 득점 비율
- 패스 성공률과 경기 결과 비교
- 주요 스탯 간 상관관계 분석

## 3. 사용 데이터

- European Soccer Database (Kaggle)
- EPL 경기 데이터
- 선수 및 팀 경기 통계 데이터(CSV 형식)

## 5. 구현 계획

본 프로젝트는 데이터 수집부터 분석 결과 시각화까지 하나의 빅데이터 파이프라인 형태로 구현한다.

### 세부 구현 과정

#### 1) 데이터 수집
Python 스크립트를 이용하여 Kaggle 및 공개 축구 데이터셋을 다운로드한다. 수집한 데이터는 CSV 형식으로 저장하며, 재실행 가능한 형태로 수집 스크립트를 작성한다.

##### 수행 내용
- 공개 축구 데이터 다운로드
- CSV 파일 정리 및 저장
- 시즌별 데이터 통합
- Python 기반 자동 수집 스크립트 작성

---

#### 2) HDFS 저장
수집한 데이터를 Hadoop Distributed File System(HDFS)에 업로드하여 분산 저장 환경을 구성한다.

##### 수행 내용
- HDFS 디렉토리 생성
- CSV 데이터 업로드
- 데이터 저장 구조 관리
- HDFS 파일 확인 및 관리

##### 사용 기술
- Hadoop HDFS
- Bash Command

---

#### 3) Hive 전처리
Hive 외부 테이블을 생성하고 SQL 기반 데이터 전처리를 수행한다.

##### 수행 내용
- Hive External Table 생성
- 결측치 제거
- 데이터 타입 변환
- 홈/원정 팀 데이터 정리
- 경기 통계 데이터 집계
- 시즌별 데이터 JOIN

##### 사용 기술
- HiveQL
- Hive External Table

---

#### 4) Spark 분석
Spark SQL과 PySpark를 활용하여 대규모 경기 데이터를 분석한다.

##### 수행 내용
- 홈/원정 승률 비교 분석
- 점유율과 승률 상관관계 분석
- 슈팅 수 대비 득점 효율 분석
- 팀별 시즌 성적 분석
- 주요 스탯 간 상관관계 계산

##### 사용 기술
- Spark SQL
- PySpark DataFrame

---

#### 5) 시각화
분석 결과를 그래프 및 차트 형태로 시각화하여 데이터 특징을 직관적으로 확인한다.

##### 수행 내용
- 팀별 승률 시각화
- 점유율 대비 승률 그래프
- 슈팅 수와 득점 관계 시각화
- 시즌별 성적 변화 분석

##### 사용 기술
- Matplotlib
- Seaborn

---

#### 6) 결과 정리 및 GitHub 관리
분석 결과 및 실행 코드를 GitHub Repository 형태로 정리한다.

##### 수행 내용
- README 작성
- 실행 방법 정리
- 코드 구조 정리
- 결과 이미지 저장
- 최종 보고서 및 발표 자료 정리
